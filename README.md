# Webpages

Collection de pages web indexées.

## Structure

```
webpages/
├── pages/          # Pages HTML
├── assets/
│   ├── css/        # Feuilles de style
│   ├── js/         # Scripts
│   └── images/     # Images et médias
└── README.md
```

## Utilisation

Chaque page est un fichier HTML autonome placé dans le dossier `pages/`.
Les ressources partagées (CSS, JS, images) sont dans `assets/`.
