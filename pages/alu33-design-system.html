01 · Foundations

Principes de design.

Quatre règles qui guident toutes les décisions visuelles Alu33.

Light-first, toujours

Fond blanc #FFFFFF ou beige warm #FAF9F7. Le dark footer sert de fermeture, jamais de base. Alternance blanc / beige pour rythmer le scroll.

Un seul accent : teal

#0A97A3 partout — CTAs, pills, accent heading, icônes. Gradient vers #07747E (dark) et #5BC4CE (light) pour la profondeur.

Une seule famille typo

Plus Jakarta Sans en 4 poids (400–700). Letter-spacing négatif sur les titres, tracking normal sur le body. Zéro exception.

Stripe polish, artisan soul

Shadows layered 3+, glassmorphism discret, transitions 200ms ease-out. Le niveau de craft de Stripe appliqué à un artisan local RGE.

14 · Copy & Tone

Voice Alu33.

Le copy est un élément de design. Le copy HTML est intouchable (travail SEO 18 mois). Liberté créative uniquement sur pills, badges et CTAs.

✓ Do

  • Copy SEO intouchable — jamais modifier le HTML fourni
  • Accent teal .xxx-accent { color: #0A97A3 } sur un mot-clé stratégique par H2
  • Phrases courtes, pas de blocs >4 lignes sans respiration
  • Verbes d'action concrets : "installer", "poser", "garantir"
  • Noms propres cités tels quels : Bordeaux, Gironde, RGE
  • Ton pro mais accessible — artisan de confiance, pas corporate
  • Liberté créative sur : pills, taglines, badges, labels, CTAs

✗ Don't

  • Modifier le copy HTML SEO sans validation
  • Italique sur headings (sauf comparison_heading-accent et comparison_payoff-heading-accent)
  • Buzzwords creux : "synergie", "disruptif", "révolutionnaire"
  • Tirets cadratins — trop AI-generated
  • Promesses quantifiées non-sourcées
  • Title case anglais sur le copy français
  • Emojis dans les titres H1/H2
